# W2D4 Assignment: The Automotive Factory

### Your Mission
Open `src/Main.java`. Notice that it is full of red error lines! This is because it is trying to use a `Car` object, but the `Car` blueprint is empty.

Open `src/Car.java` and create the correct variables (attributes) and methods (behaviors) so that `Main.java` can compile and run perfectly.
Do not modify `Main.java`!
